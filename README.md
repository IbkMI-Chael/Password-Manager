# Password Manager

A desktop app that generates, saves, and retrieves 
passwords for different websites using Python.

## Technologies Used
- Python 3
- Tkinter (built-in Python GUI library)
- JSON (built-in Python library)
- Random (built-in Python library)

## How to Run It
1. Make sure you have Python 3 installed
2. Download ALL these files into the same folder:
   - main.py
   - logo.png
3. Open your terminal in that folder
4. Run: python main.py

## How to Use It
- Enter a **website** name and **username/email**
- Click **Generate Password** for a strong random password
- Click **Add** to save the details securely
- Click **Search** to retrieve a saved password
- All passwords are stored locally in password_data.json

## Features
- Random password generator with letters, numbers and symbols
- Saves passwords locally in a JSON file
- Search and retrieve saved passwords by website name
- Confirms details before saving via a popup dialog
- Validates that no fields are left empty before saving
- Pre-fills email field automatically
